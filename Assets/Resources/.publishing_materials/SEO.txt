Управление на компьютере:
- Для выбора используйте левый клик мыши.
- Управление камерой осуществляется с помощью мыши.
- Перемещение персонажа осуществляется с помощью клавиш WASD на клавиатуре.
- Для прыжка используйте пробел (клавишу space) на клавиатуре.
- Для осмотра ножа используйте клавишу F на клавиатуре.
- Для открытия меню/настроек используйте клавишу P на клавиатуре.
- Для включения/сокрытия подсказки по управлению, используйте клавишу Tab на клавиатуре.

Управление на телефоне:
- Для выбора используйте одиночное касание.
- Управление камерой осуществляется с помощью зажатия и ведения пальца по экрану.
- Перемещение персонажа осуществляется с помощью джойстика.
- Для прыжка используйте крупную кнопку в правой части экрана.
- Для осмотра ножа используйте кнопку, расположенную над кнопкой прыжка.
- Для открытия меню/настроек используйте кнопку с изображение шестерёнки.

Computer management:
- Use the left mouse click to select.
- The camera is controlled with the mouse.
- The character is moved using the WASD keys on the keyboard.
- To jump, use the space key on the keyboard.
- To inspect the knife, use the F key on the keyboard.
- To open the menu/settings, use the P key on the keyboard.
- To enable/hide the control prompt, use the Tab key on the keyboard.

Control on the phone:
- Use a single touch to select.
- The camera is controlled by pinching and holding your finger on the screen.
- The character is moved using the joystick.
- To jump, use the large button on the right side of the screen.
- To inspect the knife, use the button located above the jump button.
- To open the menu/settings, use the button with the gear image.
